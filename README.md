# Rock Paper Scissors Game - Python

A simple command-line Rock Paper Scissors game built using Object-Oriented Programming (OOP) principles in Python.

## Project Overview

This project allows a user to play multiple rounds of Rock Paper Scissors against the computer. The game validates user input, tracks scores, determines round winners, and displays the final game result.

## Features

- Play multiple rounds against the computer
- Random computer move generation
- Input validation for user choices
- Automatic score tracking
- Final winner announcement
- Object-Oriented Programming structure
- Clear game logic using dictionaries

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Random module
- Command-line interface (CLI)
